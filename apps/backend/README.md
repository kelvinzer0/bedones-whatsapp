## Deskripsi

Repository starter TypeScript framework [Nest](https://github.com/nestjs/nest).

## Setup Proyek

```bash
$ pnpm install
```

## Kompilasi dan jalankan proyek

```bash
# pengembangan
$ pnpm run start

# mode watch
$ pnpm run start:dev

# mode produksi
$ pnpm run start:prod
```

## Jalankan tes

```bash
# tes unit
$ pnpm run test

# tes e2e
$ pnpm run test:e2e

# cakupan tes
$ pnpm run test:cov
```

## Deployment

Saat Anda siap men-deploy aplikasi NestJS Anda ke produksi, ada beberapa langkah penting yang dapat Anda ambil untuk memastikan aplikasi berjalan seefisien mungkin. Lihat [dokumentasi deployment](https://docs.nestjs.com/deployment) untuk informasi lebih lanjut.

Jika Anda mencari platform berbasis cloud untuk men-deploy aplikasi NestJS Anda, lihat [Mau](https://mau.nestjs.com), platform resmi kami untuk men-deploy aplikasi NestJS di AWS. Mau membuat deployment mudah dan cepat, hanya membutuhkan beberapa langkah sederhana:

```bash
$ pnpm install -g @nestjs/mau
$ mau deploy
```

Dengan Mau, Anda dapat men-deploy aplikasi hanya dalam beberapa klik, sehingga Anda bisa fokus membangun fitur daripada mengelola infrastruktur.

## Sumber Daya

Lihat beberapa sumber daya yang berguna saat bekerja dengan NestJS:

- Kunjungi [Dokumentasi NestJS](https://docs.nestjs.com) untuk mempelajari lebih lanjut tentang framework.
- Untuk pertanyaan dan dukungan, kunjungi [channel Discord](https://discord.gg/G7Qnnhy) kami.
- Untuk mendalami dan mendapatkan lebih banyak pengalaman praktis, lihat [kursus](https://courses.nestjs.com/) video resmi kami.
- Deploy aplikasi Anda ke AWS dengan bantuan [NestJS Mau](https://mau.nestjs.com) hanya dalam beberapa klik.
- Visualisasikan graph aplikasi Anda dan berinteraksi dengan aplikasi NestJS secara real-time menggunakan [NestJS Devtools](https://devtools.nestjs.com).
- Butuh bantuan dengan proyek Anda (paruh waktu hingga penuh)? Lihat [dukungan enterprise](https://enterprise.nestjs.com) resmi kami.
- Untuk terus mendapatkan informasi terbaru, ikuti kami di [X](https://x.com/nestframework) dan [LinkedIn](https://linkedin.com/company/nestjs).
- Mencari pekerjaan, atau punya pekerjaan untuk ditawarkan? Lihat [papan pekerjaan](https://jobs.nestjs.com) resmi kami.
